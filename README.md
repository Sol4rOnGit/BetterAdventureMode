# BetterAdventureMode
This is a minecraft mod that ensures that adventure mode is actually playable and beatable. Perfect for modpacks and usage with other mods if you want to make the objective exploration not creation.

Features in the mod:
- Tool gating (requires you to have a certain tool, e.g. an axe to mine wood)
- Specific block building (blocks placing certain blocks excluding certain blocks for a camp)
- Slower mining speed (10x slower - forces exploration)
